# Security Policy

## Reporting a vulnerability

Please report security issues privately to the maintainer instead of opening a public issue. Include a concise description, affected version or commit, reproduction steps, and impact.

## Privacy warning

Token Fence is trained from local OpenCode history. Do not publish local OpenCode histories, JSONL datasets, logs, diffs, local filesystem paths, or private tool outputs in public issues, pull requests, discussions, or examples. Redact sensitive data before sharing diagnostics.
